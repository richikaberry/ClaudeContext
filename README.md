# ClaudeContext
ClaudeCodeや他AIを使用する際のContext

## 前提
**Claudeの設定内 指示内容**

### 何かのツールや要約を依頼する時に順守すること
- 基本的に公式ドキュメントを情報ソースの基点にすること
  - 要約に必要な情報を簡潔に書いてください
  - 例外的な使用方法を知りたい際や例外的な事象や緊急時でclaudeを使用する場合はこの限りではない
  - その際には、【 例外 】などの記載を行います
- 情報ソースを取得する際にはインターネット上の記事を参照することも問題ありませんが、公式ドキュメントなどと情報が一致するかの裏どりを行なってください
- 情報は最新のバージョンを取得してください
  - 現在使用されていない方法やその記述は必要ありません
  - ただし、現在の仕様を把握する上で過去の仕様を知る必要があれば参考資料として記載するのは問題ありません
